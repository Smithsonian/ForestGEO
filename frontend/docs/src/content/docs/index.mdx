## Key Features

- **File Upload System** — CSV/TSV upload with auto-delimiter detection and per-header preview
- **Measurement Revision Upload** *(new, April 2026)* — correct specific rows in an existing census via a small CSV; multi-tab review screen before any database write
- **Unified Row Editing** *(new, May 2026)* — Impact Summary preview, typed `APPLY N` confirmation for destructive edits, per-row revert
- **In-App Site Provisioning** *(new, May 2026)* — four-step wizard creates schema, plot, census, and quadrat grid with audit and retry support
- **Automated Validation** — 14+ configurable data quality checks, including cross-census growth, dead-to-alive regression, and HOM-shift detection
- **Hard-Failure Visibility** *(April 2026)* — failed rows are no longer silently dropped; they live in `coremeasurements` with original codes preserved
- **Audit Trail** — complete change tracking; failed rows keep their ID through reingestion
- **Azure Integration** — automatic cloud backup of uploaded files
