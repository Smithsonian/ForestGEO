## Key Features

- **File Upload System** - CSV/TSV upload with auto-delimiter detection
- **Automated Validation** - 12+ configurable data quality checks
- **Failed Measurements Recovery** - Review and reingest failed records
- **Audit Trail** - Complete change tracking for compliance
- **Azure Integration** - Automatic cloud backup of uploaded files
